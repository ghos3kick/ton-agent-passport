Features

Everything an AI agent needs for verifiable on-chain identity

🔒

Soulbound Tokens (TEP-85)

Non-transferable passports bound to the agent's wallet forever. No reputation laundering, no identity markets. Written in Tact, TEP-62/85 compliant.

📊

Weighted TrustScore

40% transaction count + 60% volume, log₁₀ scaling. Backwards-compatible fallback for agents without volume data. Documented formula.

🛡️

Anti-abuse System

Dust filter (<0.01 TON), self-dealing detection, circular detection (A→agent→A within 5 min window). 8 dedicated tests.

📱

Telegram-native

Mini App with holographic passport cards, generative robot avatars, TrustScore ring. Telegram Bot for instant mint/verify/search.

🔍

Security-first

50-point security audit. HSTS, CSP, rate limiting, timingSafeEqual auth, tx mutex, input validation, SSH hardened, UFW firewall.

✅

120 Tests

36 contracts + 31 SDK + 45 Mini App + 8 Bot. TypeScript strict mode, address validation.
